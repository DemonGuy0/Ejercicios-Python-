# Ejercicios-Python-
Lista de ejercicios para resolver en Python 

Explicación corta
Listas = colecciones ordenadas.

Diccionarios = claves → valores.

Aquí entrenas manipulación de datos.

Ejercicios
1. Dada una lista de números, elimina todos los duplicados sin usar set().
2. Encuentra el segundo número más grande de una lista.
3. Rota una lista n posiciones a la derecha.
4. Aplana una lista que contiene listas adentro: [[1,2],[3,4]] → [1,2,3,4].
5. Cuenta cuántas veces aparece cada elemento (sin Counter).
6. Dada una lista de diccionarios de personas, ordénalas por edad.
7. Invierte un diccionario: claves pasan a valores y viceversa.
8. Fusiona dos diccionarios sumando valores si la clave se repite.
9. Encuentra la palabra más larga en una lista de strings.
10. Dada una matriz (lista de listas), calcula la diagonal principal.
11. Elimina de una lista todos los strings que contengan números.
12. Calcula la moda de una lista sin usar estadísticas.
13. Comprueba si dos listas son anagramas una de otra.
14. Dada una lista, crea sublistas de tamaño fijo: chunk_size.
15. Dado un diccionario de estudiantes {nombre: notas}, calcula el promedio mayor.
16. Convierte un diccionario en una lista de tuplas y ordénala.
17. Encuentra claves únicas entre dos diccionarios.
18. De una lista de números, devuelve un diccionario con: min, max, promedio.
19. Convierte una lista de números romanos a enteros.
20. Implementa una matriz identidad n×n en forma de lista.
